# HR Analytics Dashboard

This Power BI dashboard provides insights into workforce metrics,
employee distribution, and attrition trends using sample data.

## Business Objective
To support HR and leadership teams in:
- Monitoring overall workforce health
- Understanding attrition patterns
- Analyzing department-wise headcount
- Supporting data-driven HR decisions

## Key KPIs
- Total Employees
- Attrition Rate
- Average Salary
- Department-wise Headcount
- Experience Distribution
- Avg Performance Rating

## Dashboard Features
- Attrition analysis by department and gender
- Employee distribution by age and experience
- Salary trend and comparison visuals
- Interactive filters for department and location
- By Creating Salary legends Define High ,low, medium Salaries through conditional Formating

## Tools & Technologies
- Power BI Desktop
- DAX for calculated HR metrics
- Sample CSV-based HR datasets
- SQL logic for workforce aggregation (documented)

## Files in This Folder
- `HR Analytics Dashboard.pbix` – Power BI report file
- `data/hr_sample_data.csv` – Sample employee dataset
- `sql/hr_analysis.sql` – SQL queries for HR insights

⚠️ **Disclaimer:**  
All data used in this dashboard is **dummy/sample data** created solely
