# Global Product Insights and Findings Dashboard

This Power BI dashboard provides a global-level analysis of product sales
and performance across different countries and regions using sample data.

## Business Objective
To help business and leadership teams:
- Understand global sales distribution
- Identify high-performing countries and regions
- Compare regional performance across markets
- Support strategic global expansion decisions

  
  ## Data Modeling & Preparation
- Designed a **fact and dimension data model** using star schema principles
- Established relationships between sales facts and dimension tables
  (Product, Customer, Region, Date)
- Handled **large datasets** efficiently using optimized data modeling
- Integrated multiple file formats such as CSV,json,txt files
- Performed data transformations using Power Query


## Key KPIs
- Total Global Sales, Profit, Payments, users, profit%
- Total sales by Region, year, category
- Regional Contribution %
- Market Share by Region
- Year-over-Year Growth

## Dashboard Features
- Interactive map visualization for country-level analysis
- Region and country-based slicers
- Trend analysis across time periods
- Comparative KPI cards for quick insights
- KPI Selectors
- Top N Parameters
- Axis Selectors Parameters
- Dynamic Titles for KPI's

## Tools & Technologies
- Power BI Desktop
- DAX for calculated measures
- Sample CSV datasets
- SQL-based aggregation logic (documented)
